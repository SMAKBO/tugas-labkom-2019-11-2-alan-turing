Mesin Turing Universal

Dalam ilmu komputer, Mesin Turing universal adalah mesin Turing yang dapat mensimulasikan mesin Turing sewenang-wenang pada input sewenang-wenang. Mesin universal pada dasarnya mencapai ini dengan membaca deskripsi mesin yang akan disimulasikan serta inputnya dari rekamannya sendiri. Alan Turing memperkenalkan gagasan mesin seperti itu pada tahun 1936-1937.

Prinsip ini dianggap sebagai asal usul gagasan program komputer tersimpan yang digunakan oleh John von Neumann pada tahun 1946 untuk "Instrumen Komputasi Elektronik" yang sekarang menyandang nama von Neumann: arsitektur von Neumann. Setiap mesin Turing menghitung fungsi komputasi parsial tetap tertentu dari string input di atas alfabetnya. Dalam hal itu berperilaku seperti komputer dengan program tetap. Namun, kita bisa menyandikan tabel tindakan dari setiap mesin Turing dalam sebuah string. Dengan demikian kita dapat membuat mesin Turing yang mengharapkan string yang menggambarkan tabel tindakan diikuti oleh string yang menggambarkan rekaman input, dan menghitung rekaman yang dikomputasi oleh mesin Turing yang dikodekan. Turing menggambarkan konstruksi seperti itu dengan sangat lengkap dalam makalahnya tahun 1936:"It is possible to invent a single machine which can be used to compute any computable sequence. If this machine U is supplied with a tape on the beginning of which is written the S.D ["standard description" of an action table] of some computing machine M, then U will compute the same sequence as M.
